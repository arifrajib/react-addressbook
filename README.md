

React AddressBook
=============================

Using React and Redux API to manipulate and make AddressBook


### ⚠️ This is Experimental Project! ⚠️



### Usage

```
git clone https://github.com/arifrajib/react-addressbook.git
cd react-addressbook/
npm install
npm start
open http://localhost:3000

```
